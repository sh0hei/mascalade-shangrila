# mascalade-shangrila

[ShangriLa Anime API](http://api.moemoe.tokyo/) Scala wrapper library.

## Call Method

Get Cours
```Scala
val cours = Shangrila.getCours
```

Get Description
```Scala
val descriptions = Shangrila.getDescriptions(2015)
```

Get Anime
```Scala
val animes = Shangrila.getAnimes(2015, 1)
```
